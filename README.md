# wanderly
plan trips with friends, split costs without the awkward math, and discover places that actually fit your vibe — all in one place.

## features
- **schedule planning with recommendations** - plan out your trip itenerary with the help of our wanderly ai helper that provides recommendations to make the most of your trip
- **budget tracking** - set a budget goal and track shared expenses to stay within your financal means
- **live sharing** - add members to your trip, enabling synchronous access

## react setup
1. install dependencies
   ```
   npm install
   ```
2. launch locally!
   ```
   npm run dev
   ```
